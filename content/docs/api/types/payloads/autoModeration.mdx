### APIAutoModerationRuleTriggerMetadata

https://discord.com/developers/docs/resources/auto-moderation#auto-moderation-rule-object-trigger-metadata

<div className='*:border-b [&>*:last-child]:border-b-0'><div className='text-sm text-fd-muted-foreground py-4'>

<div className="flex flex-row items-center gap-4">
  <code className="text-sm">keyword_filter</code>
  <code className="text-fd-muted-foreground">{"string[]"}</code>
</div>

Substrings which will be searched for in content (Maximum of 1000)

A keyword can be a phrase which contains multiple words. Wildcard symbols can be used to customize how each string will be matched. Each keyword must be 60 characters or less
See [keyword matching strategies](https://discord.com/developers/docs/resources/auto-moderation#auto-moderation-rule-object-keyword-matching-strategies)

Associated trigger types: &lbrace;@link AutoModerationRuleTriggerType.Keyword&rbrace;, &lbrace;@link AutoModerationRuleTriggerType.MemberProfile&rbrace;



</div>
<div className='text-sm text-fd-muted-foreground py-4'>

<div className="flex flex-row items-center gap-4">
  <code className="text-sm">presets</code>
  <code className="text-fd-muted-foreground">{"AutoModerationRuleKeywordPresetType[]"}</code>
</div>

The internally pre-defined wordsets which will be searched for in content

Associated trigger type: &lbrace;@link AutoModerationRuleTriggerType.KeywordPreset&rbrace;



</div>
<div className='text-sm text-fd-muted-foreground py-4'>

<div className="flex flex-row items-center gap-4">
  <code className="text-sm">allow_list</code>
  <code className="text-fd-muted-foreground">{"string[]"}</code>
</div>

Substrings which will be exempt from triggering the preset trigger type (Maximum of 1000)

A allowed-word can be a phrase which contains multiple words. Wildcard symbols can be used to customize how each string will be matched. Each keyword must be 60 characters or less
See [keyword matching strategies](https://discord.com/developers/docs/resources/auto-moderation#auto-moderation-rule-object-keyword-matching-strategies)

Associated trigger types: &lbrace;@link AutoModerationRuleTriggerType.Keyword&rbrace;, &lbrace;@link AutoModerationRuleTriggerType.KeywordPreset&rbrace;, &lbrace;@link AutoModerationRuleTriggerType.MemberProfile&rbrace;



</div>
<div className='text-sm text-fd-muted-foreground py-4'>

<div className="flex flex-row items-center gap-4">
  <code className="text-sm">regex_patterns</code>
  <code className="text-fd-muted-foreground">{"string[]"}</code>
</div>

Regular expression patterns which will be matched against content (Maximum of 10)

Only Rust flavored regex is currently supported (Maximum of 260 characters)

Associated trigger types: &lbrace;@link AutoModerationRuleTriggerType.Keyword&rbrace;, &lbrace;@link AutoModerationRuleTriggerType.MemberProfile&rbrace;



</div>
<div className='text-sm text-fd-muted-foreground py-4'>

<div className="flex flex-row items-center gap-4">
  <code className="text-sm">mention_total_limit</code>
  <code className="text-fd-muted-foreground">{"number"}</code>
</div>

Total number of mentions (role & user) allowed per message (Maximum of 50)

Associated trigger type: &lbrace;@link AutoModerationRuleTriggerType.MentionSpam&rbrace;



</div>
<div className='text-sm text-fd-muted-foreground py-4'>

<div className="flex flex-row items-center gap-4">
  <code className="text-sm">mention_raid_protection_enabled</code>
  <code className="text-fd-muted-foreground">{"boolean"}</code>
</div>

Whether to automatically detect mention raids

Associated trigger type: &lbrace;@link AutoModerationRuleTriggerType.MentionSpam&rbrace;



</div></div>
